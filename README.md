# COLLEGE-BUS-TRACKING-AND-MANAGEMENT-SYSTEM
The College Bus Tracking and Management System is a web-based application designed to help college transportation departments efficiently manage bus routes, student allocations, and bus tracking. The system allows administrators to create and manage routes, drivers to update bus locations, and students to track their assigned bus and estimated arrival times through a user-friendly dashboard.

This project focuses on route management, role-based access control, and simulated live bus tracking to provide students with real-time updates about their bus location and arrival status.

🚀 Key Features

👨‍💼 Admin Portal:
 Create and manage bus routes,
 Assign drivers and buses,
 Allocate students to specific stops,
 Monitor system activity and bus routes

🧑‍✈️ Driver Portal

Update the current bus location (stop)

Simulate real-time bus movement along the route

🎓 Student Portal

View assigned bus route and stop

Track current bus location

See estimated arrival time (ETA)

Get real-time status updates like:

Bus is at your stop

Bus is 1 stop away

Bus has passed your stop
